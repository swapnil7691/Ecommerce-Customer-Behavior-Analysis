# Ecommerce-Customer-Behavior-Analysis

## 📊 Project Overview
E-commerce Customer Behaviour Analysis Dashboard built using Power BI.

This project analyzes customer purchasing patterns, sales performance, payment methods, delivery status, and order trends through interactive visualizations.

---

## 🚀 Key Insights Included
- Total Sales and Orders Overview
- Customers Demographics Analysis
- Category Performance Tracking
- Payment Mode Distribution
- Delivery and Order Status Performance
- City-wise Sales Analysis

---

## 🛠️ Tools Used
- Power BI
- Excel

---

## 📈 Dashboard Features
### Sales Overview
- Monthly Sales Trend
- Category Performance
- Payment Analysis

### Customer Analysis
- Sales by Gender
- Age Group Analysis
- Top Customers

### Order Performance
- Delivery Performance
- Order Status Distribution
- City and Product Filters
